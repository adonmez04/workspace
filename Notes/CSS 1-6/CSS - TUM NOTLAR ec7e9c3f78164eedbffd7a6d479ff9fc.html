What is CSS?

  • Cascading, basamakli, elden ele demek.
  • Bir ozellik, digerinin uzerinde gelirse onu baskiliyor.
  • Bir ust class’a yazilanlar, alttakileri de etkileyebiliyor, parent, child ozelligi.
  • Stillendirmek icin kullaniliyor.
  • Web sitelerinde gorunen kisimlari renklendiriyoruz, konumlandiriyoruz, stillendiriyoruz.
  • HTML ile structure olusturuyoruz.
  • CSS ile o yapiyi gorunur hale getiriyoruz. Ete kemige burunduruyoruz.
  • JS ile hareket ekliyoruz.

  • 1990 yilinda temelleri atiliyor ve gunumuze kadar geliyor.
